# krishisetu

**Krishi Setu (कृषी सेतू)** is a specialized web application designed to bridge the gap between farmers and the marketplace by providing real-time crop pricing and market information. Developed as a tool for "Market Distribution," it aims to empower the agricultural community with direct access to transparent data.

### **Core Features**
* **Real-Time Price Tracking**: The platform allows users to view current market rates for various crops, categorized into **Daily Prices** and **Seasonal Crops**.
* **Bilingual Support**: To ensure accessibility for local farmers, the entire interface supports both **English** and **Marathi**.
* **Admin Control Panel**: A dedicated administrative interface (accessible to authorized users) allows for the seamless addition and updating of crop names, prices (per kg), and categories.
* **Secure Authentication**: The app includes a integrated login and signup system powered by Firebase to manage user access.

### **Technical Specifications**
* **Frontend**: Built using **HTML5** and **Tailwind CSS** for a modern, responsive, and mobile-friendly design.
* **Interactivity**: Utilizes **Vanilla JavaScript** to handle dynamic UI updates, language switching, and tab navigation.
* **Backend & Database**: Integrated with **Firebase (Authentication and Firestore)** to provide real-time data synchronization and secure user management.
* **Visuals**: Uses **FontAwesome** for intuitive iconography and a clean, "emerald-themed" aesthetic that reflects its agricultural focus.

### **User Experience**
The application features a minimalist and professional layout, including a welcoming dashboard banner and a structured grid view for market data. It uses "Toasts" for instant feedback on user actions, such as successful price updates or login errors, ensuring a smooth and informative experience.
